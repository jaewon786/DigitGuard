# 접근성 서비스 사용 선언 (Google Play 심사용)

## 앱 이름
디짓가드 (DigitGuard)

## 접근성 서비스 사용 목적
디짓가드는 디지털 기기에 익숙하지 않은 고령층 사용자를 허위 광고, 피싱, 보이스피싱으로부터 보호하기 위해 접근성 서비스를 사용합니다.

## 구체적 기능
1. **허위 바이러스 경고 감지:** 화면에 "바이러스 감염", "핸드폰이 해킹됨" 등의 허위 광고 패턴이 표시될 때 이를 감지하고 사용자에게 경고합니다.
2. **피싱 사이트 경고:** "개인정보 유출", "계좌 정지" 등의 피싱 키워드가 감지되면 사용자에게 알립니다.
3. **보호자 알림:** 위험이 감지되면 등록된 보호자(가족)에게 자동으로 알림을 전송합니다.

## 접근성 서비스가 필수인 이유
- 허위 광고와 피싱은 주로 웹 브라우저, 앱스토어, 문자 앱 등 다양한 앱에서 발생합니다.
- 이러한 위험을 앱 간에 걸쳐 실시간으로 감지하려면 접근성 서비스를 통한 화면 텍스트 모니터링이 유일한 방법입니다.
- 대상 사용자(고령층)는 스스로 위험을 판단하기 어려우므로, 자동 감지 및 경고가 필수적입니다.

## 데이터 처리 원칙
- 화면 텍스트는 기기 내에서만 분석되며, 원문을 저장하거나 외부로 전송하지 않습니다.
- 키워드 매칭 결과만 로그로 기록되며, 보호 목적으로만 사용됩니다.
- 사용자는 언제든지 설정에서 접근성 서비스를 비활성화할 수 있습니다.

## 대상 사용자
- 60세 이상 고령층 (스마트폰 사용에 익숙하지 않은 분들)
- 해당 사용자의 가족 구성원 (보호자 역할)
