# 디짓가드(DigitGuard) 개인정보처리방침

**시행일:** 2026년 4월 16일

## 1. 수집하는 개인정보 항목

디짓가드는 서비스 제공을 위해 아래 정보를 수집합니다.

| 항목 | 수집 목적 | 보유 기간 |
|------|-----------|-----------|
| 이름 | 사용자 식별 | 회원 탈퇴 시까지 |
| 전화번호 | 긴급 연락, 보호자 연결 | 회원 탈퇴 시까지 |
| 화면 텍스트 키워드 | 허위 광고 탐지 (원문 미저장) | 분석 즉시 폐기 |
| 위험 감지 로그 | 보호 이력 제공 | 1년 |
| 앱 설치 정보 | 앱 안전성 검사 | 1년 |
| 기기 식별자 | 서비스 연동 | 회원 탈퇴 시까지 |

## 2. 접근성 서비스 사용

디짓가드는 **접근성 서비스(AccessibilityService)**를 사용합니다.

- **목적:** 화면에 표시되는 허위 바이러스 경고, 피싱 광고 등을 실시간으로 감지하여 사용자를 보호합니다.
- **수집 범위:** 화면 텍스트에서 위험 키워드 패턴만 매칭하며, 화면 전체 내용을 저장하거나 외부로 전송하지 않습니다.
- **데이터 처리:** 1차 분석은 기기 내에서 수행되며, 위험이 감지된 경우에만 키워드 매칭 결과(원문 아님)를 서버로 전송합니다.

## 3. 개인정보 제3자 제공

디짓가드는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 아래 경우는 예외합니다.

- 사용자가 사전에 동의한 경우
- 법령에 의해 요구되는 경우

## 4. 개인정보의 보호 조치

- 서버-클라이언트 간 모든 통신은 HTTPS(TLS 1.3)를 사용합니다.
- 민감 데이터는 AES-256으로 암호화하여 저장합니다.
- 접근 권한은 최소 권한 원칙에 따라 관리합니다.

## 5. 사용자 권리

사용자는 언제든지 아래 권리를 행사할 수 있습니다.

- 개인정보 열람, 수정, 삭제 요청
- 서비스 이용 동의 철회 (회원 탈퇴)
- 접근성 서비스 비활성화
