# Google Play Store 등록 정보

## 앱 이름
디짓가드 - 디지털 보호 앱

## 짧은 설명 (80자)
허위 광고, 피싱, 보이스피싱으로부터 가족을 보호하는 디지털 안전 앱

## 상세 설명

디짓가드는 스마트폰 사용에 익숙하지 않은 고령층 가족을 디지털 위험으로부터 보호합니다.

주요 기능:
- 허위 바이러스 경고 실시간 감지 및 차단
- 피싱 링크 자동 검사
- 의심스러운 앱 설치 시 보호자 알림
- 보호자 원격 모니터링 대시보드
- 긴급 연락(SOS) 원터치 기능
- 사기 예방 교육 콘텐츠

이런 분들에게 추천합니다:
- 스마트폰을 사용하는 어르신
- 부모님의 디지털 안전이 걱정되는 자녀
- 허위 광고에 자주 노출되는 분
